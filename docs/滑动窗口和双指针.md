## 滑动窗口

R++
新数从右侧进入

L++
旧数从左侧出去

L <= R 

窗口的最大值/最小值的更新结构
单调双端队列，从头到尾是从大到小的（放每个节点的位置）
每次加入的数从右边进，左边出
加入的数和头的值相比，如果小，就直接进入队列
如果大（或相等），就依次弹出，直到这个数可以进入队列
头部位置就是最大值
调整双端队列的平均复杂度是O（1）
