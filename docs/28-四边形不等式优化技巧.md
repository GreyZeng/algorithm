## 四边形不等式技巧特征

1. 两个可变参数的区间划分问题

2. 每个格子有枚举行为

3. 当两个可变参数固定一个，另一个参数和答案之间存在单调性关系

4. 而且两组单调关系是反向的：(升 升，降 降)  (升 降，降 升)

5. 能否获得指导枚举优化的位置对：上+右，或者，左+下

差(好(S左，S右))
都有O(N)的优化
