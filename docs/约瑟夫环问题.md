约瑟夫环问题

现在一共有i个节点，数到m就杀死节点，最终会活下来的节点，请返回它在有i个节点时候的编号

暴力方法: O(N*K)
可以优化到：O(N)

tips
公式，f(留下后的编号) = 杀之前的编号
号= （数-1)%i + 1
旧：1234567
新：56X1234
然后画图找规律
旧 = （新+ S-1）%i+1
S->杀的号 S = (M-1)%i+1
i->杀之前的长度

--> (新 + (m-1)) % i + 1
