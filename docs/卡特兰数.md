## 卡特兰数

k(0) = 1, k(1) = 1时，

如果接下来的项满足： k(n) = k(0) x k(n - 1) + k(1) x k(n - 2) + ... + k(n - 2) x k(1) + k(n - 1) x k(0)

或者 k(n) = c(2n, n) - c(2n, n-1)

或者 k(n) = c(2n, n) / (n + 1)

就说这个表达式，满足卡特兰数，常用的是范式1和2，3几乎不会使用到
