4. 局部最小值问题
arr 无序数组，相邻的数都不相等，返回一个局部最小的位置即可
先看0位置是不是局部最小，
然后再看N-1位置是不是局部最小
然后来中点，然后二分（某个位置比左边小，又比右最小就是那个位置）

二分取中点的安全方式 int M = L + (R - L) >> 1
N * 2 => N << 1
N * 2 + 1 => (N << 1) | 1
